## 项目经历

### MedTrust-Raft 医疗数据联盟治理平台

- 独立用 Go 实现 Raft 核心算法（Leader 选举、日志复制、心跳维持）
- 四层解耦架构：P2P 通信层 / Raft 共识层 / 区块存储层 / API 服务层
- 技术栈：Go · Raft · Vue 3 · Gin · BoltDB

### 公益众筹智能合约

- 自动化目标验证与条件执行，达标自动拨款、未达标自动退款
- 里程碑分批释放资金模型，Checks-Effects-Interactions 防重入攻击
- 技术栈：Solidity · Ethereum · Truffle · Ganache
